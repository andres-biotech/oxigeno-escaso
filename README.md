# Oxígeno Escaso | Revista Científica
> *La infraestructura de lo invisible. Divulgación biotecnológica bajo presión.*

**Sitio Web Oficial:** [oxigenoescaso.com](https://oxigenoescaso.com)

## Sobre el Proyecto
*Oxígeno Escaso* es una plataforma digital independiente nacida en Puebla, enfocada en la transferencia de conocimiento y el periodismo científico. Nuestro objetivo es desarmar los mecanismos técnicos, biológicos y físicos que operan en los límites de la realidad, traduciendo *papers* y proyectos de investigación dura (principalmente biotecnología) a una narrativa inmersiva, visceral y de alto impacto visual.

## Stack Tecnológico
Este proyecto está construido priorizando el rendimiento, la velocidad de carga y el control total del código sin depender de gestores de contenido pesados.
* **Frontend:** HTML5 Semántico, CSS3 (Vanilla), JavaScript.
* **Optimización visual:** Formato `.webp` para ultra-compresión y `.jpg` optimizado para meta-etiquetas (Open Graph).
* **Hosting & CI/CD:** GitHub Pages / Control de versiones integrado.
* **Analítica:** Google Analytics (gtag.js).

## Estructura del Repositorio

## Colaboración y Alianzas
Actualmente, la plataforma cuenta con la apertura para publicar resúmenes divulgativos de investigaciones y tesis de universidades nacionales. 

Para someter un proyecto a revisión editorial o postularse como revisor técnico/webmaster, favor de contactar a la Dirección Editorial a través del sitio web oficial. No se aceptan *Pull Requests* (PRs) con alteraciones al código sin autorización previa del equipo operativo.
